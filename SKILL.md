# 股价走势预估 Skill

## 简介

基于历史数据、技术指标、宏观环境、行业动态、公司消息，综合分析并预估A股股价未来走势。支持次日、7日、30日多周期预估。

## 核心功能

### 1. 技术面分析
- 获取股票历史K线数据（日K，60天）
- 计算技术指标：MA5/MA10/MA20/MA60
- 判断短期趋势（上涨/下跌/震荡）
- 分析支撑位和压力位
- 成交量分析（量比）

### 2. 消息面分析（六大维度）- 必须完整分析

**重要：每个维度都必须主动分析，不可遗漏！**

#### 维度1：宏观经济与政策面
- 货币政策：降准、降息、LPR调整
- 财政政策：政府债券、专项债、减税降费
- 经济数据：GDP、CPI、PPI、PMI、社融
- 高层会议：政治局、两会、部委发布会

**关键词**：央行宽松、适度宽松、财政刺激、新质生产力、内需

#### 维度2：行业与产业动态
- 产业政策：补贴、反垄断、环保限产
- 供需关系：原材料价格、产能
- 技术突破：AI、固态电池、6G
- 行业展会：车展、MWC、药交会

**关键词**：新能源车补贴、医药集采、芯片涨价、油价、金价

#### 维度3：公司微观层面
- 定期报告：年报、季报营收/利润
- 重大事项：并购重组、增持回购、合同订单
- 管理层变动
- 分红送转

**关键词**：业绩预增、大单、回购、股权激励、分红

#### 维度4：地缘政治与国际局势
- 贸易摩擦：关税、出口管制
- 战争与冲突：中东、东欧
- 外交关系：合作/制裁

**关键词**：中美关税、中东局势、油价上涨、黄金避险

**必须分析**：
- 中东冲突 → 油价 → 大宗商品/原材料成本
- 俄乌冲突 → 天然气/金属价格
- 中美关系 → 出口/供应链

#### 维度5：市场资金与情绪面
- 主力资金流向：北向资金、机构席位
- 融资融券数据
- 舆情与传闻
- 指数纳入/剔除

**关键词**：北向资金、融资买入、社保重仓、外资流出

#### 维度6：突发事件
- 自然灾害
- 公共卫生事件
- 安全事故

**关键词**：黑天鹅、突发事件

### 3. 自动获取市场环境
- 通过浏览器/新闻API获取当前市场热点
- 自动识别六大维度的实时新闻
- 结合宏观环境进行综合评分

### 4. 智能概念库
- 预置股票概念（特斯拉概念、机器人概念等）
- 支持用户补充新概念
- 自动学习保存到本地知识库

### 5. 主动学习与自主判断（重要）

**工作流程：**
1. **收到股票代码** → 自动识别相关概念
2. **多维度搜索** → 搜索股票名称 + 相关概念 + 行业动态
3. **综合分析** → 不是被动接收用户输入，而是主动输出判断
4. **验证更新** → 发现新的重要信息，主动更新到概念库
5. **不确定时** → 主动询问用户确认

**原则：**
- 不完全依赖用户提供的信息
- 用户说的不一定是正确的，需要自己验证
- 主动搜索比被动等待更可靠
- 发现新热点概念，主动添加到知识库
- **必须结合akshare-stock获取财务数据、china-stock-analysis获取估值数据**

### 5. 多周期预估

| 周期 | 消息面权重 | 影响因素 |
|------|------------|----------|
| 次日 | 100% | 技术面当日 + 消息面 |
| 7日 | 75% | MA10趋势 + 消息面 |
| 30日 | 50% | MA20趋势 + 消息面 |

## 评分体系

| 条件 | 分数 |
|------|------|
| 价格 > MA5 | +1 |
| 价格 > MA10 | +1 |
| 趋势上涨 | +2 |
| 趋势下跌 | -2 |
| 消息面偏多 | +2 |
| 消息面偏空 | -2 |

### 预估判定

| 总分 | 预估结果 | 概率 |
|------|----------|------|
| >=3 | 上涨 | 70% |
| >=1 | 震荡偏多 | 60% |
| >=-1 | 震荡 | 50% |
| >=-3 | 震荡偏空 | 40% |
| <-3 | 下跌 | 30% |

## 数据来源

- K线数据：**腾讯财经API**（优先使用）
  - 实时行情：`https://qt.gtimg.cn/q=sz002050`
  - K线历史：`https://web.ifzq.gtimg.cn/appstock/app/fqkline/get`
- 新闻数据：东方财富搜索API
  - 搜索接口：`https://search-api-web.eastmoney.com/search/jsonp`
- 市场环境：通过新闻API自动获取

**必须结合使用的技能**：
1. **akshare-stock**：获取A股行情、财务数据、板块信息
2. **china-stock-analysis**：获取公司基本面、估值数据、财务分析

**注意**：新浪财经API已被WAF拦截，优先使用腾讯财经接口

## 注意事项

1. 预估仅作参考：股市有风险
2. 消息面影响随时间递减
3. **六大维度必须完整分析，不可遗漏地缘政治**
4. 数据源优先使用腾讯财经（新浪已被WAF拦截）
5. **必须主动搜索相关概念**，而非仅搜索股票名称
6. **必须结合akshare-stock、china-stock-analysis技能获取更多数据**

## 智能概念库工作流

收到股票代码后：
1. **识别相关概念**：义乌、外贸、退税商店、跨境电商等
2. **多维度搜索**：
   - 搜"股票名称"
   - 搜"相关概念"（如"义乌"、"退税"、"外贸"）
   - 搜"行业动态"
3. **综合分析**：结合所有相关新闻判断影响
