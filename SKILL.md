# 股价走势预估 Skill

## 简介

基于历史数据、技术指标、宏观环境、行业动态、公司消息，综合分析并预估A股股价未来走势。支持次日、7日、30日多周期预估。

## 核心功能

### 1. 技术面分析

使用 **akshare-stock** 获取数据，计算以下指标：

| 指标类型 | 指标名称 | 说明 |
|----------|----------|------|
| 均线 | MA5/MA10/MA20/MA60 | 判断趋势方向 |
| 趋势 | 上涨/下跌/震荡 | MA排列状态 |
| 支撑/压力 | 支撑位、压力位 | 价格上下限 |
| 量能 | 量比、成交量 | 放量/缩量 |
| RSI | 相对强弱指数 | 超买超卖（14日） |
| MACD | 指数平滑异同移动平均线 | 金叉/死叉 |
| 布林带 | Bollinger Bands | 上下轨突破 |

### 2. 消息面分析（六大维度）

**重要：每个维度都必须主动分析，不可遗漏！**

#### 维度1：宏观经济与政策面
- 货币政策：降准、降息、LPR调整
- 财政政策：政府债券、专项债、减税降费
- 经济数据：GDP、CPI、PPI、PMI、社融
- 高层会议：政治局、两会、部委发布会

**关键词**：央行宽松、适度宽松、财政刺激、新质生产力、内需

#### 维度2：行业与产业动态
- 产业政策：补贴、反垄断、环保限产
- 供需关系：原材料价格、产能
- 技术突破：AI、固态电池、6G
- 行业展会：车展、MWC、药交会

**关键词**：新能源车补贴、医药集采、芯片涨价、油价、金价

#### 维度3：公司微观层面
- 定期报告：年报、季报营收/利润
- 重大事项：并购重组、增持回购、合同订单
- 管理层变动
- 分红送转

**关键词**：业绩预增、大单、回购、股权激励、分红

#### 维度4：地缘政治与国际局势
- 贸易摩擦：关税、出口管制
- 战争与冲突：中东、东欧
- 外交关系：合作/制裁

**关键词**：中美关税、中东局势、油价上涨、黄金避险

**必须分析**：
- 中东冲突 → 油价 → 大宗商品/原材料成本
- 俄乌冲突 → 天然气/金属价格
- 中美关系 → 出口/供应链

#### 维度5：市场资金与情绪面
- 主力资金流向：北向资金、机构席位
- 融资融券数据
- 舆情与传闻
- 指数纳入/剔除

**关键词**：北向资金、融资买入、社保重仓、外资流出

#### 维度6：突发事件
- 自然灾害
- 公共卫生事件
- 安全事故

**关键词**：黑天鹅、突发事件

### 3. 智能概念库

- 预置股票概念（特斯拉概念、机器人概念等）
- 支持用户补充新概念
- 自动学习保存到本地知识库

### 4. 主动学习与自主判断

**工作流程**：
1. **收到股票代码** → 自动识别相关概念
2. **多维度搜索** → 搜索股票名称 + 相关概念 + 行业动态
3. **综合分析** → 不是被动接收用户输入，而是主动输出判断
4. **验证更新** → 发现新的重要信息，主动更新到概念库
5. **不确定时** → 主动询问用户确认

**原则**：
- 不完全依赖用户提供的信息
- 用户说的不一定是正确的，需要自己验证
- 主动搜索比被动等待更可靠
- 发现新热点概念，主动添加到知识库

### 5. 多周期预估

| 周期 | 技术面权重 | 消息面权重 | 影响因素 |
|------|------------|------------|----------|
| 次日 | 60% | 40% | 技术面当日 + 消息面 |
| 7日 | 50% | 50% | MA10趋势 + 消息面 |
| 30日 | 40% | 60% | MA20趋势 + 消息面 |

## 评分体系

| 条件 | 分数 |
|------|------|
| 价格 > MA5 | +1 |
| 价格 > MA10 | +1 |
| MA5 > MA10（金叉） | +2 |
| MA10 < MA20（死叉） | -2 |
| RSI > 70（超买） | -1 |
| RSI < 30（超卖） | +1 |
| 消息面偏多 | +2 |
| 消息面偏空 | -2 |

### 预估判定

| 总分 | 预估结果 | 概率 |
|------|----------|------|
| >=3 | 上涨 | 70% |
| >=1 | 震荡偏多 | 60% |
| >=-1 | 震荡 | 50% |
| >=-3 | 震荡偏空 | 40% |
| <-3 | 下跌 | 30% |

## 数据来源

### 主要数据源 - 必须优先使用

**akshare-stock** - 用于获取：
- A股实时行情
- 历史K线数据（日K/周K/月K）
- 财务数据（营收、利润、负债率）
- 板块信息
- 资金流向（北向资金、融资融券）

**china-stock-analysis** - 用于获取：
- 公司基本面分析
- 估值数据（PE、PB、PS）
- 财务健康状况
- 内在价值评估
- 行业对比

### 辅助数据源 - 备用

当主数据源不可用时，使用以下接口：

| 用途 | 接口 | URL |
|------|------|-----|
| 实时行情 | 腾讯财经 | `https://qt.gtimg.cn/q=sz002050` |
| K线历史 | 腾讯财经 | `https://web.ifzq.gtimg.cn/appstock/app/fqkline/get` |
| 新闻搜索 | 东方财富 | `https://search-api-web.eastmoney.com/search/jsonp` |

### 新闻数据

通过东方财富搜索API获取：
- 公司新闻
- 行业动态
- 宏观经济新闻
- 地缘政治相关

## 智能概念库工作流

收到股票代码后：
1. **识别相关概念**：义乌、外贸、退税商店、跨境电商等
2. **多维度搜索**：
   - 搜"股票名称"
   - 搜"相关概念"（如"义乌"、"退税"、"外贸"）
   - 搜"行业动态"
3. **综合分析**：结合所有相关新闻判断影响

## 注意事项

1. 预估仅作参考：股市有风险
2. 消息面影响随时间递减
3. **六大维度必须完整分析，不可遗漏地缘政治**
4. **数据源必须优先使用akshare-stock和china-stock-analysis**
5. **必须主动搜索相关概念**，而非仅搜索股票名称
6. 只有当主数据源不可用时，才使用辅助接口
7. RSI<30视为超卖，RSI>70视为超买
